# CIFI Farm Mission Optimizer

A browser-based version of the CIFI Farm Mission Optimizer.

This site is designed to run as a simple static website hosted including:
- Optimizer
- FAQ
- Credits and Changelog

## Features

- Browser-based optimizer
- Farm mission selection
- Campaign targeting and focus
- Resource priority controls
- Ouroboros-aware UI and mission visibility
- Local save/load using browser storage
- Reset to built-in defaults

## Project Files

- `index.html` — page structure
- `styles.css` — site styling
- `script.js` — optimizer logic and browser persistence
